步骤

1. CoreMark 上板跑分

  1. 在主机确认产物为 RISC-V ELF： 
    $ file coremark.exe
# 应含 UCB RISC-V / RISC-V 字样
$ sha256sum coremark.exe
  2. SCP 上传到荔枝派 4A（用户名与 IP 换成你的板子）： 
    $ scp coremark.exe user@board-ip:/tmp/
$ ssh user@board-ip 'sha256sum /tmp/coremark.exe'
# 与主机 sha256sum 对照，两端一致
  3. SSH 登录板端，赋予执行权限并运行： 
    $ ssh user@board-ip
$ chmod +x /tmp/coremark.exe
$ /tmp/coremark.exe
  4. 从输出中抄录 CoreMark 分数（常见字段为 iterations/sec 或 CoreMark 总分）。完整终端输出另存为日志文件。
排错：板端报 Exec format error 时，回主机用 file 核对是否误用了主机 GCC；再核对 SCP 路径与权限。命令细节见讲义 1.2 / 1.3。

字段你的记录
产物文件名

CoreMark 分数

2. 通道与工具链自证

每条保留一行真实输出。

# 板端：SSH 与架构
$ ssh user@board-ip 'hostname; uname -m'
# uname -m 应为 riscv64

# 主机：工具链三元组
$ riscv64-unknown-linux-gnu-gcc -dumpmachine
# 应为 riscv64-unknown-linux-gnu
