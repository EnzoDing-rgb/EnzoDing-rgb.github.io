# 如意 RISC-V Linux 实践课 · 课程评价标准

课题组自评用。全文四部分、共 12 条；每条 0–10 分；**不算**「合格 / 优秀」标签，只保留分数。

本标准是课题组自定的**统一口径**。结构上对应 CIPP 的四层思路；学习成效条目写法吸收国内课程中常见的 OBE「可观察目标 + 证据」做法。来源链接见文末。

---

## 计分规则

1. 四部分等权，各占 **25%**。
2. 每部分 **3 条**，部分内等权：该部分得分 = 三条分数的算术平均。
3. 课程总分 = 四部分得分的算术平均（满分 10）。
4. 评分人：课题组自评。

**公式（便于核对）**

- 部分得分 \(P_i = (a_{i1} + a_{i2} + a_{i3}) / 3\)
- 总分 \(T = (P_1 + P_2 + P_3 + P_4) / 4\)

## Reference（来源）

本标准**不照搬**下列框架的全部条文，只取其思路：

### CIPP（四层结构来源）

- 斯塔弗尔比姆（D. L. Stufflebeam）提出的 CIPP：背景（Context）、输入（Input）、过程（Process）、成果（Product）。中文概述：[国外教育评价模式演进及启示（宁波工程学院高教所）](https://gjs.nbut.edu.cn/info/1052/1513.htm)
- 课程评价语境下的 CIPP 四步说明：[课程评价模式（中公网校）](https://www.eoffcn.com/kszx/detail/1008991.html)
- 国内程序设计课用 CIPP 建指标体系的示例：[基于 CIPP 模型的程序设计课程质量评价体系构建](http://www.cxcybjb.com/CN/Y2024/V7/I24/5)

### OBE（成效条目写法来源）

- OBE（成果导向）常用四问：要学生取得什么成果、为什么、如何帮助学生取得、如何知道已取得。中文说明：[OBE 教育模式详解（海南热带海洋学院评建材料）](https://pgw.hntou.edu.cn/shpgw/zcwx/202304/P020230419649596614419.pdf)
- 国内嵌入式类课程用「课程目标 + 考核证据 + 达成判断」的实践示例：[嵌入式系统与应用课程目标达成度评价与持续改进](https://xuebaosk.ahut.edu.cn/cn/article/pdf/preview/10.3969/j.issn.1671-9247.2025.05.019.pdf)
- 产出导向课程评估与 CIPP 结合的讨论：[基于 CIPP 评价模式的产出导向课程评估指标体系构建](https://www.hanspub.org/journal/paperinformation?paperid=32366)
